How this works
Edits save to the server (/api/content) automatically and become visible to visitors within seconds — no file upload step. For images, click the Upload button next to any image URL field to upload a file directly, or paste a URL from Unsplash, Drive, Cloudinary, etc. Export data.js downloads a local backup; Reset discards your local edits and reloads from the server.
